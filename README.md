# Blockchain Transaction Analysis Portfolio

This repository contains blockchain transaction audit reports created using publicly available blockchain data.

The purpose of this project is to demonstrate practical skills in:

- Transaction tracing
- Wallet flow analysis
- Hop analysis
- Source of funds analysis
- Risk assessment

Networks Analyzed:

- XRP Ledger (XRPL)
- Bitcoin
- Ethereum (planned)

Tools Used:

- Blockchain.com Explorer
- XRPSCAN
- Public blockchain data

---

## Reports

Reports will be added as analyses are completed.

---

## Methodology

Each transaction is analyzed using the following process:

1. Transaction identification
2. Source of funds analysis
3. Hop analysis
4. Fee analysis
5. Risk assessment
6. Final conclusion
